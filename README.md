## fullstack-node-react  

### Technologies used:
* nodejs 
* reactjs
* apollo
* postgresql
* jwt - jsonwebtoken
* typeorm
* argon2

### Service the application

** Backend **  

> Run watch for converting typescript to backend   
` yarn watch `

> Run nodemon for watching the application  
` yarn serve `

** Fronend **  
> Run serve the reactjs application  
` yarn start `
